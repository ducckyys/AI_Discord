# Roadmap

The roadmap reflects intended direction, not a delivery guarantee.

## Next

- First-class PostgreSQL schema and migration workflow
- Configurable context retention and conversation summarization
- Role and channel allowlists for dedicated AI channels
- Improved operational diagnostics for LM Studio availability

## Future

- Optional admin dashboard for server settings
- Metrics suitable for self-hosted observability
- Streaming Discord responses where platform constraints allow
- Additional provider adapters behind the existing provider interface, without changing the LM Studio-first default
