# Roadmap

[English](ROADMAP.md)

Roadmap ini menunjukkan arah pengembangan yang diinginkan, bukan jaminan waktu rilis.

## Berikutnya

- Skema PostgreSQL dan alur migrasi kelas satu
- Retensi konteks yang dapat dikonfigurasi serta peringkasan percakapan
- Allowlist role dan channel untuk channel AI khusus
- Diagnostik operasional yang lebih baik untuk ketersediaan LM Studio

## Mendatang

- Dashboard admin opsional untuk pengaturan server
- Metrik yang sesuai untuk observabilitas self-hosted
- Respons Discord streaming bila batasan platform memungkinkan
- Adapter provider tambahan melalui antarmuka provider saat ini tanpa mengubah default LM Studio-first
