## Mulai cepat

1. Baca README utama berbahasa Indonesia di root repositori.
2. Konfigurasikan environment dengan `.env.example`.
3. Jalankan LM Studio sebelum menjalankan bot.
4. Opsional: jalankan SearXNG bila ingin memakai pencarian.

## Catatan

- Bot berfungsi sebagai asisten Discord yang didukung instance LM Studio lokal Anda.
- SearXNG bersifat opsional dan tidak diperlukan untuk chat dasar.
- Jika hanya memerlukan chat, konfigurasi SearXNG dapat dibiarkan kosong atau tidak digunakan.
